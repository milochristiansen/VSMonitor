Commands:
There are several special commands for the monitor, everything else is sent to the server. These commands are:
  • :recover: Start a currently down game server.
  • :server create \"name\" [stable|unstable|x.x.x.x]: Create a new server.
  • :server update [x.x.x.x]: Update the current server (does not work in the monitor tab).
  • :kill (monitor|server): Shutdown the monitor or kill a misbehaving game server.
  • :user (create|delete) "name": Create or delete a user.
