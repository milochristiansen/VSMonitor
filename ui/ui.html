Buttons:
  • ☼: Settings, set your API token here.
  • >>>: Send command to server, does the same thing as pressing "enter".
Commands:
There are three special commands for the monitor, everything else is sent to the server. These commands are:
  • :recover: Starts the game server.
  • :kill (server|monitor): Shutdown the monitor or kill a misbehaving game server.
  • :token (issue|revoke) username Issue or revoke the token for the given username. Administrator only.
